# html-form
# OMÜ Etkinlik Başvuru Formu

Bu proje, web programlama dersi için hazırladığım temel bir HTML form ödevidir. Amacı, HTML form etiketlerini kullanmayı öğrenmek ve form verilerini bir test adresine (webhook) göndermektir.

## Projede Neler Var?
* Kullanıcıdan metin, tarih, dosya (CV) ve renk gibi farklı türde bilgileri alan HTML5 form kutucukları.
* Hatalı girişi engelleyen basit kontroller (Örneğin; e-posta formatı zorunluluğu, telefon numarası şablonu).
* JavaScript ile yapılmış, kaydırdıkça anlık değişen basit bir not ortalaması (GANO) çubuğu.

## Nasıl Çalıştırılır?
Herhangi bir program kurmanıza gerek yoktur. Projeyi bilgisayarınıza indirin ve `index.html` dosyasını Chrome, Edge veya Firefox gibi herhangi bir tarayıcıda çift tıklayarak açın.
